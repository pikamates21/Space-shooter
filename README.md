# Space-shooter
Space shooter hra, jako můj pololetní projekt do programování

Projekt je dělán v pythonu, lavně přes pygame. 

Ovládá se šípkami vpravo a vlevo, nebo klávesami 'a' nebo 'd',
střílí se pak mezerníkem.

Cílem je získat co nejvyšší skóré s tím, že se nepřátelé objevují
čím dál častěji.
